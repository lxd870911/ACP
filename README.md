## 云帮 是一款基于容器技术，以应用为中心的PaaS，借助它您可以：

- #### 对接现有CI/CD系统
> 实现快速开发、部署与发布。

- #### 作为容器管理平台
> 以应用为中心，高效管理容器和底层计算资源。

- #### 作为运维平台
> 实现服务快速伸缩、微服务治理、业务性能监控。

<br>
## 快速开始
您可以访问 [平台安装](平台安装.md) 章节阅读云帮社区版的安装手册，快速部署一套属于自己的私有云平台。

如果您想深入了解云帮平台，可以访问 [设计思想](设计思想.md)，[架构](架构.md)和 [组件说明](组件说明.md) 等章节详细阅读。
